技术栈

后端
Python 3.11+ · FastAPI · SQLite（sqlalchemy）
akshare（数据源） · APScheduler（定时）
numpy/pandas（指标计算）
前端
纯 HTML + CSS + JavaScript（SPA）
无框架依赖，轻量快速
Fetch API 调用后端 REST 接口
💡 前端选纯前端框架-Free的方案，方便部署和分享。一个 Python 进程搞定全部，部署到任何 VPS/Docker 都极其简单

整体架构长这样：数据层(akshare) → 策略引擎(indicator计算+战法筛选) → 服务层(FastAPI+SQLite) → 前端SPA
